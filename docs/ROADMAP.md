# Roadmap - Mu Trading Bot

## Released milestones

- v0.1.0 - Scanner + watchlist + decisions.
- v0.2.0 - Basic backtesting.
- v0.3.0 - Dashboard summary + scheduler improvements.
- v0.4.0 - Maintenance tools.
- v0.5.0 - Portfolio documentation.
- v0.6.0 - S&P 500 import + scanner batching.
- v0.6.1 - Scheduler pre-close confirmation fix.

## Next improvements

- Public deployment demo.
- Web dashboard frontend.
- Advanced performance metrics.
- Backtesting with commissions and slippage.
- Exportable reports.
- Improved USA holidays and half-day handling.
- Optional Telegram integration.
- Better fundamentals support.
- Better traceability for decisions and signals.

## Non-goals

- No automatic buying or selling.
- No broker order execution.
- No profitability promises.
- No financial advice.
