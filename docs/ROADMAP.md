# Roadmap - Mu Trading Bot

## Released milestones

- v0.1.0 - Scanner + watchlist + decisiones.
- v0.2.0 - Backtesting basico.
- v0.3.0 - Dashboard summary + scheduler corregido.
- v0.4.0 - Maintenance tools.
- v0.5.0 - Documentacion y demo portfolio.

## Next improvements

- Deploy demo.
- Dashboard web frontend.
- Metricas avanzadas de performance.
- Backtesting con comisiones y slippage.
- Reportes exportables.
- Mejor soporte para feriados USA y half-days.
- Integracion futura con Telegram.
- Mejoras de fundamentals.
- Mejor trazabilidad de decisiones y seńales.

## Non-goals

- No compra automatica.
- No ejecucion de ordenes en broker.
- No promesas de rentabilidad.
- No asesoramiento financiero.
