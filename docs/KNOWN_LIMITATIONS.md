# Known Limitations

- Uses `yfinance` and public market data, which can be delayed, incomplete, rate-limited, or unavailable.
- Technical scanner is rule-based, not predictive AI.
- Backtesting is basic and does not fully model slippage, commissions, spread, liquidity, partial fills, or execution quality.
- No automatic order execution.
- Discord notifications depend on webhook availability and network connectivity.
- Market calendar and timezone handling is simplified. Official USA holidays and half-days are not fully modeled.
- Fundamental analysis is intentionally limited in the current version.
- Educational and portfolio use only.
